# Tampines Rat Watch 🐀

A browser-based game set in Tampines, Singapore. Bust rats before they overrun the town!

## Play

Open `index.html` in any modern browser — no build step, no dependencies.

Or play it live via GitHub Pages (once enabled in repo settings).

## Features

- Single HTML file — fully self-contained
- Mobile-friendly (portrait/landscape support, fullscreen mode)
- Upload your own background photo
- Local leaderboard
